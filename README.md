# c-multiclient-chat-server

A simple multi-client chat application written in C using TCP sockets and POSIX threads.

## Features
- Client-server architecture
- Multiple clients supported
- Real-time message broadcasting
- Private messaging using @username
- Thread-safe communication using mutexes
- Runs on localhost

## Technologies
- C (POSIX)
- TCP sockets
- pthreads
- Linux system calls
